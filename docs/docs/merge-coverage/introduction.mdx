Combines code coverage between different javascript test runners, like:
- Jest
- Vitest
- Playwright
- Cypress.


## Usage
```sh-session
$ npm install -g @jtmdias/merge-coverage
$ merge-coverage COMMAND
running command...
$ merge-coverage (--version)
@jtmdias/merge-coverage/1.1.0 linux-x64 node-v20.16.0
$ merge-coverage --help [COMMAND]
USAGE
  $ merge-coverage COMMAND
...
```
