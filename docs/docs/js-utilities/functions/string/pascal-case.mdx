---
title: pascalCase
---
Formats the given string in pascal case fashion

## API

```typescript
function pascalCase(str: string): string;
```

### Usage

```typescript
import { pascalCase } from '@jtmdias/js-utilities';

pascalCase('hello world')
//-> 'HelloWorld'

pascalCase('va va boom')
//-> 'VaVaBoom'
```
